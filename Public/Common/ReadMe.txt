Common库
    注：
        Common库将在服务器与客户端公用
        默认使用.NetFramework 2.0版本，如需升级，务必通知所有人

        Data        加载全局、场景资源
        Event       事件系统、命令机制
        Geometry    几何、数学工具
        Scene       地图元素类定义，Scene接口
        Entity      游戏角色类接口
        Tool        工具类，CD工具类、数据加载工具、操作返回值即提示管理、状态管理类
